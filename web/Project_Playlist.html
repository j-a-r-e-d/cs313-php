ROAD-TRIP PLAYLIST MAKER

This application was created to help you create a playlist of your favorite songs for a road-trip. Elements which are still missing:
Cannot yet store selected songs into the playlist in the database.
Cannot choose to add to a currently selected playlist.
User cannot see nor select from list of playlists they've already created.
No login security features have been implemented yet.
Cannot delete users or playlists yet.
Cannot edit a playlist.
Total playback time is not calculated for the playlist yet.
